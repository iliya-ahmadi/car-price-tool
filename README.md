Car Price Tool (Divar)

A simple web tool that fetches real car prices from Divar listings and shows a quick market overview.

This project focuses on simplicity and usability — just enter the car information and instantly see market prices.


---

Features

• Search car prices from Divar
• Clean Streamlit web interface
• Persian city selection
• Fast price extraction
• Lightweight and minimal design


---

Tech Stack

Python
Streamlit
Requests
BeautifulSoup


---

Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/car-price-tool.git

cd car-price-tool

Create virtual environment:

python -m venv venv

Activate environment (Windows):

venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py


---

Usage

1. Enter car name


2. Select city


3. Choose production year


4. Click Analyze Price


5. View extracted market prices
